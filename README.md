## Setup Locally
**Prerequisite**
- 1. NodeJs
- 2. Git

1. Clone the project

    ```
    git clone https://github.com/amanrajrana/RobustKey-PasswordGenerator.git
    ```

2. Go to the project directory
    ```
    cd RobustKey-PasswordGenerator
    ```

3. Install Dependencies
    ```
    npm install
    ```

4. Start App
    ```
    npm run dev
    ```
